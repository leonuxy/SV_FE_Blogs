- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework
- [ShadCN UI](https://ui.shadcn.dev/) – Beautifully designed components
- [Lucide Icons](https://lucide.dev/) – Icon set for modern UIs
